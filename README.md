# 💸 Bill Split

A modern web application for splitting bills between friends and groups. The goal is to make it easy to calculate each person's share, track payments, and simplify expense management.

## ✨ Features

* Create and manage bills
* Add participants
* Split bills equally or by custom amounts
* Track who paid and who owes
* View bill history
* Responsive and user-friendly interface

## 🛠️ Tech Stack

### Frontend

* Angular
* TypeScript
* HTML5
* CSS3

### Backend (Planned)

* ASP.NET Core Web API
* Entity Framework Core
* SQL Server

## 🚀 Getting Started

### Prerequisites

* Node.js 16+
* Angular CLI 14+
* npm

### Installation

```bash
git clone https://github.com/FaresGomaa1/bill-split.git
cd bill-split
npm install
ng serve
```

Open your browser and navigate to:

```
http://localhost:4200
```
